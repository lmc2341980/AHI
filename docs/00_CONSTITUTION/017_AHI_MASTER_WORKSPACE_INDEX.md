# Purpose

This document defines the Master Workspace Index (MWI) of the Artificial Hybrid Intelligence (AHI) ecosystem.

The Master Workspace Index is the single approved entry point for every Human, Artificial Hybrid Intelligence, Robot and Agent participating in an AHI Workspace.

Its purpose is to enable every participant to reconstruct the complete working environment without depending on previous chat sessions or personal memory.

GitHub repositories are the operational Source of Truth.

# Responsibilities

The Master Workspace Index shall:

* describe the current Workspace;
* identify all active repositories;
* locate all approved documents;
* define the current project milestone;
* identify active discussions;
* identify pending work;
* identify responsible roles;
* provide the navigation structure for the entire ecosystem.
