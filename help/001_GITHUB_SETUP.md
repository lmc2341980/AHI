# GitHub Repository Setup

**Document ID:** HELP-20260704-001

---

# Purpose

This document defines the standard procedure for creating any repository in the AHI ecosystem.

---

# Standard Settings

## Repository Name

Use PascalCase with hyphens.

Examples:

- AHI
- AHI-Core
- AHI-Studio
- AHI-Lang

---

## Visibility

Choose:

- Public
- Private

according to AHI Governance.

---

## Description

Write one concise sentence explaining the repository purpose.

---

## Website

Leave blank unless an official website exists.

---

## Initialize Repository

Always enable:

- README

Choose:

- License

Leave empty:

- .gitignore

unless a programming language has already been selected.

---

## License

Default:

Apache License 2.0

unless another license has been approved.

---

## Topics

Add GitHub Topics.

Example:

- ahi
- artificial-hybrid-intelligence
- ai
- knowledge

---

# Required Files

Every repository should gradually include:

README.md

LICENSE

docs/

help/

ai/

---

# Completion

After repository creation:

1. Create folders.
2. Add README.
3. Commit.
4. Push.
5. Notify the project owner.
