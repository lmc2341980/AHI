# Pre-Commit Checklist

**Document ID:** HELP-20260704-010

---

# Purpose

Verify that every change follows AHI standards before committing.

---

# Repository

- Correct repository selected.
- Correct visibility.

---

# Folder

- Correct folder.
- Correct hierarchy.

---

# Document

- Document ID exists.
- Status exists.
- Version exists.
- Repository Owner exists.
- Layer exists.

---

# Markdown

- Title exists.
- Sections are complete.
- Formatting is valid.

---

# Commit Message

Starts with:

AHI:

Clearly describes the change.

---

# Review

- Self-review completed.
- AI validator passed.
- Reviewer approved.

---

# Push

- Commit pushed successfully.
- GitHub displays latest version.

---

# Final Question

Before every commit ask:

> Does this change improve the AHI ecosystem?

If the answer is **No**, revise before committing.
