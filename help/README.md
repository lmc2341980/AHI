# AHI HELP

**Document ID:** HELP-20260704-000

## Purpose

The HELP directory is the operational manual for humans working within the AHI ecosystem.

Unlike `docs/`, which defines standards and architecture, the `help/` directory provides practical step-by-step instructions.

---

# Target Users

- Founder
- Developers
- Contributors
- Organizations
- AI Operators
- Community Members

---

# Relationship

```
docs/
    Defines WHAT.

help/
    Explains HOW.

ai/
    Automatically verifies compliance.
```

# Principle

If you do not know how to perform a task, check the corresponding HELP document before creating new content.

GitHub is the source of truth.

HELP is the operational manual.

AI must also follow these standards.
